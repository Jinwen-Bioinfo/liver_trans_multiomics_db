# GSE13440 Curation Note

## Why This Dataset Matters

GSE13440 is the first independent liver-allograft biopsy expression layer added after GSE145780. It helps address the circular-evidence concern because it does not reuse INTERLIVER cluster labels. The clinical comparison is narrower and more confounded:

| Standardized state | Original label | Sample count | Interpretation |
| --- | --- | ---: | --- |
| `ACR` | ACR-predominant with superimposed RHC | 9 | Acute cellular rejection predominant, with recurrent hepatitis C context |
| `RHC_no_ACR` | RHC with no ACR | 13 | Recurrent hepatitis C without acute cellular rejection |

## Ingestion

Metadata is generated by:

```bash
python scripts/ingest_geo_series.py GSE13440
```

Gene-level expression evidence is generated by:

```bash
python scripts/build_gene_evidence.py GSE13440
python scripts/build_signature_scores.py GSE13440
```

## Expression Layer

- Platform: `GPL1291`
- Gene-level mappings: 14,538 gene symbols
- Observed mapped probes: 19,255
- Contrast: `ACR_vs_RHC_no_ACR`
- Method: gene-level probe averaging followed by Welch two-sample t-test and Benjamini-Hochberg FDR within the contrast

## Caveats

- This is not a clean TCMR-versus-stable comparison. The control group is recurrent hepatitis C without ACR.
- Some immune markers have substantial missingness on this platform.
- The contrast supports independent evidence exploration, not standalone diagnostic performance.
