# Reviewer Dashboard

This dashboard is the shortest single place to understand the current state of LiverTx-OmicsDB as a resource.

## What It Aggregates

The dashboard pulls together:

- resource metadata
- release/citation/contact posture
- quickstart journeys
- reviewer walkthroughs
- QC/provenance status
- NAR readiness summary

## What A Reviewer Should Be Able To Answer Quickly

After reading the dashboard, a reviewer should be able to answer:

1. what the current version is
2. whether the resource is public yet
3. how to start evaluating it
4. what the strongest demonstrator questions are
5. how much processed/QC/provenance coverage already exists
6. what still blocks a NAR-facing release

## Current Honest Reading

Right now the dashboard should communicate a balanced message:

- the scientific framing is strong
- the question-driven evidence organization is real
- the processed coverage is substantial
- the deployment/release infrastructure still lags behind the curation work

That combination is acceptable for an advanced local prototype, as long as it stays explicit.
