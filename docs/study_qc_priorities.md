# Study QC Priorities

This note records which studies should receive the first dedicated study-level QC views.

## First Priority Cohorts

The first per-study QC pages should focus on the strongest current anchors:

1. `GSE145780`
2. `GSE243887`
3. `GSE200340`
4. `PXD046355`

## Why These Four

These cohorts are the best starting set because together they cover:

- graft biopsy bulk RNA
- donor-organ RNA
- blood/PBMC monitoring RNA
- direct donor-liver bile proteomics

They also map cleanly onto the highest-value demonstrator questions already exposed in the resource.

## What The First QC Pages Should Show

For each study, the QC page should eventually show:

- sample count and state distribution
- processed artifact coverage
- provenance coverage
- major label caveats
- what the dataset can and cannot currently support

## Why This Matters

The resource now has reviewer-facing status layers for release, tutorials, and QC/provenance coverage. The next step is to bring that same explicitness down to the study level.
