# Resource Status

This page is the shortest reviewer-facing summary of where LiverTx-OmicsDB stands as a resource.

## Release

Status: `in_progress`

Current state:

- current version is `0.1.0`
- the portal and API work locally
- the project is not yet deployed at a stable public URL

What is already true:

- question-first workflows exist
- anonymous API access exists
- release/citation/contact posture is documented

What is still missing:

- stable reviewable URL
- public domain
- DOI-backed release snapshot

## Help and Tutorials

Status: `in_progress`

Current state:

- help section exists in the portal
- quickstart journeys exist
- glossary exists
- reviewer walkthrough exists

What is still missing:

- a tighter reviewer-facing tutorial sequence in the live product
- a more complete first-time user glossary/FAQ experience
- direct in-product surfacing of all tutorial materials

## QC and Provenance

Status: `planned`

What is still missing:
