# GSE11881 Curation Note

## Why This Dataset Matters

GSE11881 is the first blood-based liver transplant expression layer in LiverTx-OmicsDB. It supports the non-invasive monitoring use case by comparing peripheral blood mononuclear cells from operationally tolerant and non-tolerant liver transplant recipients.

| Standardized state | Original label | Sample count | Interpretation |
| --- | --- | ---: | --- |
| `operational_tolerance` | tolerant liver transplant recipient | 9 | Recipient with operational tolerance |
| `non_tolerant` | non-tolerant liver transplant recipient | 8 | Recipient requiring ongoing immunosuppression |

## Ingestion

Metadata is generated by:

```bash
python scripts/ingest_geo_series.py GSE11881
```

Gene-level expression evidence is generated by:

```bash
python scripts/build_gene_evidence.py GSE11881
python scripts/build_signature_scores.py GSE11881
```

## Expression Layer

- Platform: `GPL570`
- Gene-level mappings: 22,878 gene symbols
- Observed mapped probes: 45,782
- Contrast: `operational_tolerance_vs_non_tolerant`
- Method: probe-to-gene averaging followed by Welch two-sample t-test and Benjamini-Hochberg FDR within the contrast

## Caveats

- This is PBMC, not graft tissue.
- Source values are preserved as source-normalized Affymetrix microarray intensity. Fold-change approximation is therefore not reported unless a study is explicitly on a log2 scale.
- The current database uses this as a blood immune-monitoring evidence layer, not as a graft rejection classifier.
